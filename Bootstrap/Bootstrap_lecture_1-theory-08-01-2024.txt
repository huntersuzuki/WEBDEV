Bootstrap Framework
===================

Need of Framework
================
Framework helps you to create your application faster.
They provide ready made components to build your application.

What is bootstrap?
=================
bootstrap is a front end CSS framework.
It contains pre defined css classes in a css file that is complied.[Error Free].
It Provides grid structure to place element accurately at a particular position on the 
web page.
Bootstrap has 12 column grid structure.
It is a project of Twitter.
History of Bootstrap: Pending

Documentation : what is boostrap?
Bootstrap is a powerful, feature-packed frontend toolkit.
 Build anything—from prototype to production—in minutes.

Webpage layout 
==============
Arrangement of website components[navbar,footer,images,content]
on the web page is called as Webpage layout.

Breakpoint: It is a screen width at which the website or webpage layout changes.

Avaliable breakpoint 

extra small        no infix          <576       range: 0 to < 576px
small              sm                >=576      range: >=576 to <768px
medium             md                >=768      range: >=768 to <992 px
large              lg                >=992      range: >=992 to <1200 px 
extra large        xl                >=1200     range: >=1200 to <1400 px
extra extra large  xxl               >=1400     range: >=1400 and above
